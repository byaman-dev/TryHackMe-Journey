# 📖 About This Module

Every cybersecurity professional works with computers every day, but protecting a system requires more than simply knowing how to use one. Understanding how computers are built, how they process information, how operating systems interact with hardware, and how modern computing environments are structured provides the technical foundation required for every area of cybersecurity.

This module introduces the essential building blocks of modern computer systems. It explores different types of computers, internal hardware components, client-server architecture, virtualization, and cloud computing. These concepts help explain how modern digital infrastructure operates and why cybersecurity professionals must understand both hardware and software before defending them.

Whether investigating a compromised workstation, securing a cloud environment, or deploying virtual machines for penetration testing, these fundamental concepts appear throughout a cybersecurity career.

# 🌍 Why Computer Fundamentals Matter

Cybersecurity is built on understanding systems before securing them.

Attackers exploit weaknesses in hardware, operating systems, applications, cloud platforms, and virtual environments. Without understanding how these technologies function, identifying vulnerabilities and implementing effective security measures becomes significantly more difficult.

Computer fundamentals provide the technical knowledge required for:

* System Administration
* Digital Forensics
* Malware Analysis
* Cloud Security
* Network Security
* Endpoint Protection
* Incident Response
* Security Engineering
* Penetration Testing

Every advanced cybersecurity topic builds upon these foundational concepts.

# 🧠 Core Concepts

Throughout this module I explored the following concepts.

---

## Computer Types

Different computing devices are designed for different purposes.

Examples include:

* Desktop Computers
* Laptop Computers
* Servers
* Smartphones
* Tablets
* Embedded Systems
* Internet of Things (IoT) Devices
* Supercomputers

Each category differs in processing power, storage, scalability, portability, and intended use.

---

## Internal Computer Components

A computer consists of multiple hardware components working together.

Major components include:

* Central Processing Unit (CPU)
* Random Access Memory (RAM)
* Motherboard
* Storage Devices (HDD & SSD)
* Graphics Processing Unit (GPU)
* Power Supply Unit (PSU)
* Network Interface Card (NIC)
* Cooling Systems

Each component performs a specialised role in processing, storing, or transmitting information.

---

## Client-Server Architecture

Modern computing relies heavily on the client-server model.

A client requests services or information, while a server processes requests and provides responses.

Examples include:

* Web Browsers ↔ Web Servers
* Email Clients ↔ Mail Servers
* Mobile Apps ↔ Cloud Services
* Enterprise Workstations ↔ Domain Controllers

This architecture allows resources to be shared efficiently across multiple users and devices.

---

## Virtualisation

Virtualisation allows multiple virtual machines (VMs) to operate on a single physical computer through a hypervisor.

Benefits include:

* Resource optimisation
* Isolation
* Testing environments
* Rapid deployment
* Disaster recovery
* Safe malware analysis

Virtualisation is widely used by cybersecurity professionals for penetration testing, digital forensics, and secure laboratory environments.

---

## Cloud Computing

Cloud computing provides computing resources over the Internet instead of relying solely on local hardware.

Common cloud service models include:

* Infrastructure as a Service (IaaS)
* Platform as a Service (PaaS)
* Software as a Service (SaaS)

Major cloud providers include:

* Microsoft Azure
* Amazon Web Services (AWS)
* Google Cloud Platform (GCP)

Cloud computing has transformed how organisations deploy, manage, and secure digital infrastructure.
