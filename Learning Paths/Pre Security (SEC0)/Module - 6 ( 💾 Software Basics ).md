# 📖 About This Module

Software is the driving force behind every modern computer system. From operating systems and web browsers to cloud services, mobile applications, and cybersecurity tools, software enables hardware to perform meaningful tasks.

For cybersecurity professionals, understanding software extends beyond simply using applications. It involves understanding how programs are developed, how they execute instructions, how data is stored and processed, and where vulnerabilities can arise throughout the software lifecycle.

This module introduces the fundamental concepts of software, programming, software development, databases, version control, and application security. These concepts provide the knowledge required before studying secure software development, web security, malware analysis, cloud computing, and penetration testing.

By understanding how software is designed and operates, cybersecurity professionals are better equipped to identify weaknesses, secure applications, and respond to software-related security incidents.

# 🌍 Why Software Matters

Every digital service relies on software.

Examples include:

* Operating Systems
* Mobile Applications
* Banking Systems
* Social Media Platforms
* Web Applications
* Artificial Intelligence
* Cloud Services
* Enterprise Software
* Cybersecurity Tools

Because software controls how systems behave, attackers frequently exploit programming mistakes, insecure configurations, and design flaws to compromise systems.

Understanding software enables defenders to identify vulnerabilities before attackers exploit them.

# 🧠 Core Concepts

Throughout this module I explored the following concepts.

---

## What is Software?

Software is a collection of programs and instructions that tell a computer how to perform specific tasks.

Unlike hardware, software is intangible and can be modified, updated, and improved over time.

Examples include:

* Windows
* Linux
* Microsoft Office
* Google Chrome
* Visual Studio Code
* Python
* Wireshark
* Nmap

---

## Types of Software

Software can generally be divided into two categories.

### System Software

System software manages hardware and provides a platform for applications.

Examples:

* Operating Systems
* Device Drivers
* Firmware
* Utility Programs

---

### Application Software

Application software performs tasks for end users.

Examples:

* Web Browsers
* Office Applications
* Games
* Messaging Applications
* Database Systems
* Security Tools

---

## Programming

Programming is the process of writing instructions that computers can execute.

Programming languages allow developers to solve problems through logical sequences of commands.

Common programming languages include:

* Python
* Java
* C
* C++
* JavaScript
* Go
* Rust
* PowerShell
* Bash

Programming plays an important role in automation, scripting, security tooling, malware analysis, and cloud administration.

---

## Software Development Life Cycle (SDLC)

Modern software is developed through a structured process known as the Software Development Life Cycle.

Typical stages include:

* Planning
* Requirements Analysis
* Design
* Development
* Testing
* Deployment
* Maintenance

Security should be considered throughout each stage, rather than added after development is complete.

---

## Databases

Databases store and organise structured information.

Most web applications rely on databases to manage user accounts, transactions, application settings, and business information.

Common database systems include:

* MySQL
* PostgreSQL
* Microsoft SQL Server
* SQLite
* MongoDB

Understanding databases provides the foundation for studying SQL, database security, and common attacks such as SQL Injection.

---

## Version Control

Version control systems allow developers to track changes made to source code over time.

Benefits include:

* Change tracking
* Collaboration
* Rollback capability
* Branching
* Code review

Git is the most widely used version control system.

GitHub provides cloud-based collaboration and project hosting for software development.

---

## Software Security

Software security focuses on designing and developing applications that resist attacks.

Important principles include:

* Secure Coding
* Input Validation
* Authentication
* Authorization
* Encryption
* Error Handling
* Logging
* Regular Updates

Building security into software from the beginning reduces the likelihood of vulnerabilities.
